Aérodynamisme

Icône Nom Description Marque favorable Cumul ?
Aérodynamisme AérodynamismeRend tes super sauts plus rapides.
Leviathus		 Oui

Description

Il est possible de réduire le temps de super saut du personnage. Cela permet d'accélérer le retour sur le front de bataille ou bien de fuir une situation tendue.
C'est aussi très utile pour se déplacer de beacons en beacons sur les très grandes maps comme Tours Girelle. Ce bonus réduit un peu le temps de préparation avant le saut et surtout le temps passé dans les airs.
